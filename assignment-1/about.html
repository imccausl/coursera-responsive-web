About this Website

Here you are at the about page! This is the about page, which is pretty self-explanatory as to its purpose. But if you're still confused, read on! You'll find some information about this website here.

Hey, so this page is about stuff and stuff. The purpose of this page is to give you some information about this website. So here is that information: basically it presents a task list because, well, that's what the assignment told me to do. We haven't really learned how to make checkboxes so I used an ordered list instead. Also, I couldn't really be bothered, even if I used checkboxes to make the checkboxes actually do anything. At least not yet. It just seemed a little bit too above-and-beyond what is was necessary for this assignment. So you know? I hope this about page was extremely super informative.

Oh yeah, one last thing about this page. I decided to use some bootstrap glyphicons because I felt like it made the whole thing a little bit more intuitive, especially for more visual people. So yeah, I did that on purpose thinking about the user-experience and about giving the user visual clues as to where they are and where they can go by using the menu. You will also find that it is fully responsive thanks to the awesome power of bootstrap. Cool!
